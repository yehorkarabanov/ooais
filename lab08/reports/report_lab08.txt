– What is the difference between a flattened image vector and an image tensor?
A flattened image vector is a one-dimensional array that contains all the pixel values of an image, while an image tensor is a multi-dimensional array that can represent the spatial structure of the image (e.g., height, width, and color channels).

– What does a PyTorch Dataset do?
A PyTorch Dataset is a class that provides an interface for accessing and manipulating a dataset. It allows you to define how to load and preprocess the data, and it can be used in conjunction with a DataLoader to efficiently iterate over the dataset during training.

– What does a DataLoader do?
A DataLoader is a PyTorch class that provides an efficient way to load and iterate over a dataset. It can handle batching, shuffling, and parallel loading of data, which can help improve the performance of training a model.
